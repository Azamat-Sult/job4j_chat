# job4j_chat

В данном проекте реализован REST API для чата

GET /person/ список всех пользователей.
GET /person/{id} - пользователь с id.
POST /person/ - создает пользователя.
PUT /person/ - обновляет пользователя.
DELETE /person/ - удаляет.
